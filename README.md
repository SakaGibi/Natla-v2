# Natla v2

Natla v2 is a refactored version of the original Natla P2P voice chat application.

## Why v2?
The original P2P mesh architecture reached its scalability limits, especially in larger rooms, causing connection instability and audio degradation.

## Planned changes
v2 introduces a server-assisted architecture (SFU) to improve:
- Scalability
- Connection stability
- Media quality under load

## Current Status
Refactoring in progress. Core architecture is being redesigned.


***This version is not production-ready yet.***
